# Comparison Memo

1. Compare the options across the agreed dimensions
2. Name the strongest overall choice, the best budget option, and the biggest caution
3. Tie the recommendation back to the target use case and actual evidence
4. Call out where more testing is still needed
5. Post the final memo to Recommendations and the short verdict to Status
